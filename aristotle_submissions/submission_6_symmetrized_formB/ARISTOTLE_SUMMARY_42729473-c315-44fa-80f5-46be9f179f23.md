## Key Finding: The Original Killing Tensor Formula Was Incorrect

The original submission defined the Killing tensor as:

**K_{μν} = r² g_{μν} + Δ Σ δ^r_μ δ^r_ν** ← INCORRECT

I performed extensive numerical verification (using Float arithmetic to evaluate the symmetrized covariant derivative at generic points) and confirmed that this formula does **not** satisfy the symmetrized Killing tensor equation ∇_{(α} K_{μν)} = 0. For example, at (M=1, a=0.5, r=3, θ≈π/4), the symmetrized equation gives values as large as 269.5, far from zero.

## Corrected Killing Tensor

The correct Killing tensor for the Kerr spacetime (associated with the Carter constant) is:

**K_{μν} = -a² cos²θ · g_{μν} + Σ² · (dθ)_μ(dθ)_ν + sin²θ · ω³_μ ω³_ν**

where ω³ = a dt - (r²+a²) dφ. Equivalently, using the Kinnersley principal null covectors ℓ and n (with ℓ·n = -1):

**K_{μν} = Σ(ℓ_μ n_ν + ℓ_ν n_μ) + r² g_{μν}**

Both forms were verified numerically to satisfy the symmetrized Killing equation to machine precision.

## What Remains (sorry)

The main theorem `symmetrized_killing_equation` has one `sorry`. Each of the 64 index cases (α,μ,ν ∈ {0,1,2,3}) reduces to a rational function identity in (r, sinθ, cosθ, M, a). The trivial cases (where α∈{0,3}, so all partial derivatives vanish) were verified to close with `norm_num; ring`. The harder cases (α∈{1,2}) involve extremely large rational expressions that exceed the computational budget of automated tactics like `field_simp; ring`.

A structural proof via the Killing-Yano tensor would be more elegant: the KY 2-form f satisfies ∇_{(α}f_{β)γ} = 0, and K = f·g⁻¹·f automatically satisfies the Killing tensor equation by an algebraic identity independent of the specific metric. However, formalizing this requires tensor calculus infrastructure not currently available in Mathlib.
